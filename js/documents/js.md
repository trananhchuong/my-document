
### Types:
  #### There are 2 types:
    - Primitive data:
        - Number
        - String
        - Boolean
        - Undefined
        - Null
        - Symbol 

    - Complex data:
        - function
        - object 

### Sync / Async

    Sync: code chạy từ trên xuống dưới

    Async: không biết khi nào tác vụ mới thực hiện xong, nên js sinh ra callback để giải quyết

    => callback hell
